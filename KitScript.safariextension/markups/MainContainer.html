Paneltip

The Global Settings panel lets you enter global exclusion patterns for the interaction of your user scripts with Web pages.

Global exclusion URL patterns have the highest precedence over all the other integration layers.

There's three integration layers: The Global Layer (this panel), The User Layer (setted on a User Script Settings panel) & The Script Layer (entered in the user script header).

Paneltip

User Settings

After the Global Layer, the second layer of user script integration is the User Layer. Set the exclusions and inclusions in their respective lists in this tab.

Remember that inclusions patterns always have a lower precedence than exclusion patterns.

Script Settings

The Inclusion and Exclusion patterns of the Script Layer are listed in their respective listboxes on your left and they can be inversely upgraded to the second layer, The User Layer (script include to user exclude and script exclude to user include).

Script Editor

You can edit or paste and update a User script in the text area on your left. The user script settings are updated on each "Update" button click.

Remember that a user script is identified by its metadata @name + @namespace. If two user scripts end up having the same name and namespace, a data conflict can arise.
